The course project is dedicated to the development of a web application for interactive raster image processing using computer graphics algorithms with the ASP.NET web framework and the C# and JavaScript programming languages.
The developed web application is designed for creating, editing, and publishing graphical images. The system’s capabilities include drawing tools, basic image processing algorithms, and user interaction through authorization mechanisms.

A relational database management system, SQLite, was used to store user data, while the Microsoft Azure cloud platform was employed for storing graphic files. 
During development, the Component Object Model (COM) technology was applied to implement individual graphics processing modules, and the Docker containerization platform was used. 
The project was developed in the MS Visual Studio environment using the .NET platform and HTML/CSS technologies to create the user interface.
